## 快捷键
+ 可以在keyboards shortcuts里面去调整
- 全部折叠 ctrl + k, ctrl + 0
- 折叠当前代码段 ctrl + shift + [
+ 通过方向键，添加多个光标 CTRL(SHIFT) + ALT + UpArrow
